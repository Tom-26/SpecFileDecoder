Конвертер спектрофотометрических данных

Описание программы

Данный питоновский скрипт преобразует бинарные файлы спектрофотометра в формат CSV. Программа автоматически определяет структуру данных (формат чисел с плавающей запятой или целочисленный формат) и генерирует ось длин волн для каждого значения. Результатом работы является CSV-файл, который можно открыть в таблицах или специализированных аналитических программах.

Требования для работы
	•	Python: Рекомендуется версия 3.6 или выше.
	•	Tkinter: Библиотека tkinter используется для создания графического интерфейса. В большинстве сборок Python она устанавливается по умолчанию.
	•	Операционная система: Windows

Установка и подготовка
	1.	Установка Python:
Скачайте и установите Python с официального сайта: python.org.
	2.	Проверка tkinter:
Откройте командную строку (CMD) и выполните команду:

python -m tkinter

Если откроется окно с информацией, tkinter установлен корректно.

	3.	Получение файлов:
Передайте заказчику два файла:
	•	Основной скрипт: sp_trans.py
	•	Этот файл с инструкцией (например, как README.txt)

Как пользоваться программой
	1.	Запуск программы:
Запустите файл sp_trans.py двойным щелчком (если файлы .py ассоциированы с Python) или выберите «Открыть с помощью Python».
	2.	Выбор файлов:
При запуске откроется диалоговое окно для выбора одного или нескольких файлов спектрофотометра. Выберите нужные файлы и нажмите «Открыть».
	3.	Обработка файлов:
Программа автоматически обрабатывает выбранные файлы. В случае возникновения ошибок будет показано сообщение об ошибке, а после успешного завершения обработки всех файлов появится сообщение об успешном выполнении.
	4.	Результат:
Для каждого обработанного файла в той же папке создается новый CSV-файл с таким же именем, но с расширением .csv.

Возможные ошибки и рекомендации
	•	Отсутствие выбранных файлов:
Если файлы не выбраны, программа покажет информационное сообщение и завершится. Повторите запуск и выберите нужные файлы.
	•	Ошибки обработки файла:
Убедитесь, что файл является корректным бинарным файлом спектрофотометра и не поврежден.
	•	Проблемы с Python или tkinter:
Проверьте, что установлена совместимая версия Python и что библиотека tkinter доступна.
	•	Резервное копирование:
Рекомендуется создавать резервную копию исходных данных перед обработкой.

Если возникнут дополнительные вопросы, обращайтесь к разработчику.
